02 / Problem

The annoying blinds.

UC San Diego students living in dorms, bedrooms, and shared spaces typically use manually operated chain loop blinds which are left in one position all day. This causes monitor glare, excessive heat, or rooms too dark that rely on artificial lighting.

03 / Existing Solutions

What is already out there.

Each existing option works (partially).

Commercial smart blinds

Strengths

  • Simple remote operation
  • Polished consumer experience

Limitations

  • Requires replacing existing blinds
  • Rarely truly autonomous

Roller shades

Strengths

  • Smooth diffusion of light
  • Effective glare reduction

Limitations

  • Binary — open or closed
  • Fabric degrades faster
